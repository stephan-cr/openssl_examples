================
OpenSSL examples
================

-------------
Prerequisites
-------------

* gcc_
* cmake_ (>= 2.8.0)
* openssl_ (>= 1.0.0)

.. _gcc: http://gcc.gnu.org
.. _cmake: http://www.cmake.org
.. _openssl: http://www.openssl.org

--------
Building
--------

::

  % cd <checkout-directory> ; mkdir build ; cmake .. ; make

--------
Examples
--------
